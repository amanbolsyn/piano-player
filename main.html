Piano Player

Virtual piano that has two modes of operation.

  1. Interactive mode

    Records and responds to user real times interaction. Records played piano sequences and exports it in a seperate JSON file.

  2. Prepared mode:

    Plays pre-recorded piano sequences with form JSON file. Has playback controls such as stop, pause, speed adjustments.
